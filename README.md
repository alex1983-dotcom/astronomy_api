# Astronomy API

**Исследовательский/образовательный проект** — REST API на FastAPI для изучения паттернов управления астрономическими данными. Построен на SQLAlchemy 2.0 (асинхронный ORM), PostgreSQL и валидации через Pydantic.

⚠️ **Это учебный проект** — не предназначен для реальных астрономических приложений.

## Возможности
- CRUD-операции для небесных тел (звёзды, планеты, галактики), астрономов и наблюдений
- Расширенная фильтрация и поиск с валидацией через Pydantic
- Асинхронная работа с PostgreSQL (`asyncpg`) через SQLAlchemy 2.0
- Заготовка JWT-аутентификации
- Автоматическая документация OpenAPI (Swagger UI)
- Эндпоинты статистики для практики анализа данных

## Цель проекта
Создан для изучения:
- Современных паттернов FastAPI + SQLAlchemy 2.0 (асинхронность)
- Сложных связей в БД (иерархии небесных тел, связи многие-ко-многим)
- Дизайна API для научных данных
- Эволюции схем БД через Alembic

## Не для продакшена
- Нет валидации реальных астрономических данных
- Упрощённые физические/астрономические модели
- Обучающая структура кода (подробные комментарии, примеры)
